# Task Manager App

A simple Task Manager web application that allows users to create, read, update, and delete tasks with priority levels. Data is saved using browser localStorage.

## Features
- Add, edit, delete tasks
- Set task priority (High, Medium, Low)
- Data persistence using localStorage
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
